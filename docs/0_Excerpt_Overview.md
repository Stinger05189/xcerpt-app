# 0_Excerpt_Overview.md

## The Vision

**Xcerpt** is a premium, desktop-based Context Staging Environment designed specifically for developers who integrate AI models (ChatGPT, Claude, Gemini, etc.) into their daily workflows.

Acting as a robust IDE for AI context management, Xcerpt allows developers to manage multiple codebases simultaneously. Xcerpt solves context friction by acting as an intelligent middle-layer. It allows developers to visually curate files, surgically "compress" large files, manage multiple independent presets within parallel workspaces, and seamlessly drag-and-drop optimized results directly into their AI chat interfaces.

## Core Value Propositions

1. **Frictionless LLM Integration:** Curate context visually and drag the final optimized package straight from Xcerpt into your browser.
2. **Context Compression:** Save thousands of tokens and improve AI focus by visually skipping unneeded code blocks while preserving structural integrity.
3. **Ephemeral Quick Exports:** Instantly estimate token counts for specific selections and generate lightning-fast, process-bound temporary payloads without touching global export configs.
4. **Workspace Organization & Presets:** Never lose a context setup. Manage multiple independent Presets (e.g., "UI Bug Context" vs "Database Context") within a single implicitly saved workspace.
5. **AI Filter Bypassing:** Seamlessly bypass chat application file limits via Automated Chunking, and bypass arbitrary file-type rejections using Global Extension Overrides.

## Key Features

- **Implicit Auto-Saving & Named Workspaces:** Open the app and start working immediately. Every session is automatically saved in the background.
- **The Workspace Browser:** A centralized hub to query your entire history of workspaces based on project root paths.
- **Multi-Workspace Tabs:** Open entirely different workspaces simultaneously via the consolidated application header.
- **Workspace Presets:** Maintain multiple independent exclusion rules and compression states within the same codebase.
- **Quick Selection Stats & Exports:** Paint files in the tree to view live token estimates, then explicitly generate and drag ephemeral payloads.
- **Unified File Tree:** A highly performant file tree that displays _all_ files. Excluded files are visually dimmed rather than hidden.
- **Monaco-Powered Compression Engine:** A built-in code editor lets you highlight code blocks and mark them to be skipped.
- **Global Theming & Configurations:** Customize application colors, fonts, and scale via CSS variables. Define custom extension overrides (e.g., mapping `.uproject` to `.json`) to ensure compatibility with strict AI upload filters.
- **Smart Export & `ExportedFileTree.md`:** Xcerpt dynamically generates a highly optimized markdown representation of your project structure, annotating skipped files, compressed blocks, and extension overrides, giving the AI perfect spatial awareness.
